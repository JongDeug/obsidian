[man]
명령어 핸드북
ex :
	man -k 검색어 (검색))

[ls]
디렉토리 내용 리스트

[cat]
파일 합치기, 표준 출력에 대한 출력
ex : 
	cat > file (덮어쓰기)
	cat >> file (추가)

	cat < file (읽기)
	cat < input.txt > output.txt (읽기 후 출력)

[|]
파이핑 명렁어
ex : 
	date | tee date.txt | cut --delimiter=" " --fields=2 > month.txt

[pwd]
현재 경로

[cd]
파일 이동

[touch]
파일 생성
ex : 
	touch file1.txt
	touch file1.txt file2.txt
[mkdir]
폴더 생성
ex : 
	mkdir seoul
	mkdir -p korea/busan (폴더 안 폴더)
	mkdir "gangnam gumi" (띄어쓰기)

[rm]
삭제
ex : 
	rm file.txt (파일)
	rm *.txt
	rm -r seoul (폴더)
	rm -r -i korea (안전하게 삭제)

[cp]
복사
ex : 
	cp what? where?
	cp file1.txt seoul (파일)
	cp -r seoul gumi (폴더)

[mv]
이동, 이름 변경
ex : 
	mv what? where?
	mv file1.txt file5.txt (파일 이름 변경)
	mv seoul gumi (폴더 이름 변경)
	mv file1.txt gumi (파일 이동)
	mv seoul gumi (폴더 이동)

[nano]
파일 생성, 수정

[find]
파일, 폴더 찾기
(레벨, 타입, 이름, 사이즈)
ex : 
	find . -type f -size +100k -size -5M -exec cp {} copy_here
	.(위치) 100k이상  5M이하 모든 파일을 copy_here 폴더에 복사

[grep]
파일 내용 검색
ex : 
	grep h file.txt (파일에서 h 문자 찾기)
	grep -c h file.txt (라인 카운트)
	grep -i a file.txt (대소문자 구분 X)
	grep -i "the same day" file.txt (문장 검색)
	grep -v was file.txt (was가 포함되지 않은 내용 출력)
	
[tar, gzip, bzip]
아카이빙, 압축
ex : 
	tar -cvf archive.tar file*.txt (tar 파일 생성, -v 진행상황 -f 파일)
	tar -xvf archive.tar (tar 파일 압축 풀기)

	gzip archive.tar (gunzip)
	bzip2 archive.tar (bunzip2)

	zip zipfile.zip file*.txt

	tar -cvzf archive.tar.gz file*.txt (gzip 파일 생성 -z gzip)
	tar -cvjf archive.tar.bz2 file*.txt (bzip2 파일 생성 -j bzip2)

	tar -xvzf archive.tar.gz (gzip 압축 풀기)
	tar -xvjf archive.tar.bz2 (bzip2 압축 풀기)

[bash script]
ex : 
	1. nano script.sh
	2. 파일 상단에  #!/bin/bash
	3. 실행 명령어 작성
		mkdir ~/Desktop/seoul
		cd ~/Desktop/seoul
		touch file{1..100}
		ls -l ~/Desktop/seoul > ~/Desktop/seoul.log
	4. bash script.sh

[Crontab]
Scheduled Automation

ex : 
	crontab -e
	*	 *	    *        *        *         command
	분(0-59) 시간(0-23) 일(1-31) 월(1-12) 요일(0-7) 명령어

	* * * * * echo "Hello Linux" >> ~/Desktop/hello.txt (매분마다 실행)
