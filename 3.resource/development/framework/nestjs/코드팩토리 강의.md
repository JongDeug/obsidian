
## NestJS 란?

NestJS는 자체적으로 서버 아키텍처를 제공해준다. 그래서 테스트하기 쉽고, 디커플링이 잘 돼있고, 유지보스가 편한 서버를 제작하게 해준다.

## NestJs 시작

`sudo npm i -g @nestjs/cli`
`nest new foldername`
